Hardware (Componentes físicos)

  1. Unidad Central de Procesamiento (CPU)
    • Es el “cerebro” de la computadora
    • Ejecuta instrucciones y procesos
  2. Memoria RAM
    • Memoria de trabajo temporal
    • Guarda datos mientras la computadora está encendida
  3. Disco duro o unidad de estado sólido (HDD/SSD)
    • Almacena archivos, programas y el sistema operativo
    • El SSD es más rápido que el HDD
  4. Placa base (Motherboard)
    • Conecta todos los componentes entre sí
    • Tiene ranuras para RAM, CPU, tarjetas, etc.
  5. Fuente de poder
    • Transforma la energía eléctrica para alimentar la computadora
  6. Tarjeta gráfica (GPU)
    • Procesa imágenes y gráficos
    • Es esencial para juegos y diseño
  7. Tarjeta de sonido (opcional)
    • Controla el audio (aunque muchas placas ya la tienen integrada)
  8. Unidad óptica (DVD/CD) (opcional)
    • Para leer discos (cada vez menos usada)
  9. Ventiladores y sistema de enfriamiento
    • Mantienen la temperatura adecuada del sistema
  10. Gabinete o carcasa
    • Contiene y protege todos los componentes internos

Periféricos (Entrada/Salida)

Entrada

  • Teclado
  • Ratón o mouse
  • Micrófono
  • Escáner
  • Cámara web

Salida

  • Monitor o pantalla
  • Impresora
  • Bocinas o audífonos

Entrada/Salida

  • Pantallas táctiles
  • Dispositivos USB (almacenamiento, etc.)
  • Lector de tarjetas

Software (Componentes lógicos)

  1. Sistema operativo (Ej.: Windows, macOS, Linux)
    • Administra todos los recursos de hardware y software
  2. Controladores (Drivers)
    • Permiten que el sistema reconozca e interactúe con el hardware
  3. Aplicaciones o programas
    • Word, Excel, navegadores web, juegos, etc.
  4. BIOS/UEFI
    • Programa básico que inicia al encender la computadora y configura el hardware
  5. Antivirus y herramientas de seguridad
